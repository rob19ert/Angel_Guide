# Role
Ты Senior React Developer. Твоя задача — внедрить интерактивную карту в приложение "Справочник Рыболова".

# Task
Реализовать компонент карты, отображающий водоемы Москвы и Московской области, используя бесплатные технологии.

# Tech Stack & Libraries
*   **Map Provider:** OpenStreetMap (OSM).
*   **Library:** `react-leaflet` и `leaflet`.
*   **Styling:** Tailwind CSS.

# Requirements

1.  **Dependencies:**
    *   Установи `react-leaflet` и `leaflet`.
    *   Не забудь импортировать CSS Leaflet'а в проект, иначе карта развалится.

2.  **Data Source:**
    *   Используй существующий массив `MOCK_LAKES`.
    *   Убедись, что у каждого объекта в массиве есть координаты `coordinates: [lat, lon]`. Если их нет — добавь реалистичные координаты для водоемов Московской области (например, для Озера Сенеж, Истринского водохранилища и т.д.). Центр карты поставь на Москву `[55.75, 37.57]`.

3.  **Visual Style (Pixel Art Adaptation):**
    *   Стандартная карта OSM выглядит слишком современно. Примени CSS-фильтры к слою `TileLayer`, чтобы придать ей ретро-вид. Например: `filter: sepia(20%) contrast(110%) grayscale(40%)`.
    *   **Контейнер:** Карта должна быть в "рамке" стиля приложения: `border-4 border-black`, жесткие тени.
    *   **Маркеры:** Стандартные синие пины Leaflet не подходят. Создай кастомную иконку (`L.icon`), используя твой существующий ассет (например, `fishIcon` или уменьшенную версию `mini_lake.jpg`), или просто нарисуй CSS-квадрат, чтобы это выглядело как метка на RPG-карте.

4.  **Functionality:**
    *   При клике на маркер должен открываться **Popup**.
    *   В Popup'е: Название озера, мини-картинка и кнопка "Подробнее", ведущая на страницу `LakeInfoPage` (`/lakes/:id`).
    *   Кнопка в попапе должна быть стилизована под твои желтые пиксельные кнопки.

5.  **Integration:**
    *   Добавь этот компонент на страницу `LakesPage` (например, переключатель "Список / Карта") ИЛИ создай отдельный роут `/map`.
